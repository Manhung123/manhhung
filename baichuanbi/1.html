Câu lệnh điều kiện còn được gọi là câu lệnh ra quyết định (decision making) • Cho phép thay đổi luồng thực thi của chương trình • Lựa chọn thực thi một khối lệnh dựa trên việc đánh giá một điều kiện cho trước • JavaScript hỗ trợ các câu lệnh điều kiện ìf và switc-case câu lệnh ìf cú pháp là ìf (condition) { // one or more statements; } câu lệch switc-case cú pháp switc-case switch (greeting) { case "hello": console.log("You are speaking English"); break; case "hola": console.log("Usted está hablando español"); break; case "bonjour": console.log("Vous parlez français"); break; case "hallo": console.log("Du sprichst Deutsch"); break; }
